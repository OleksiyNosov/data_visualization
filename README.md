# Requirements
- Python 3.6.5

# Setup
## Install

```sh
sudo easy_install pip

sudo pip install --upgrade virtualenv

virtualenv --system-site-packages -p python3 .

pip install matplotlib

pip install pandas
```

## Start

```sh
source ./bin/activate
```

## Stop

```sh
source deactivate
```

# Check

```sh
python3 -V
```
